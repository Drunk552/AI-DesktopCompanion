# AI-DesktopCompanion 项目目录结构

```
AI-DesktopCompanion/
├── docs/                              # 项目文档
│   ├── 开发文档.md                      # 开发计划与阶段说明
│   └── 目录结构.md                      # 本文件
│
├── tools/                             # 外部工具集（Windows 端）
│   ├── ffmpeg/                        # FFmpeg 工具链
│   │   ├── ffmpeg.exe                 # 音视频编解码
│   │   ├── ffplay.exe                 # 播放器
│   │   └── ffprobe.exe                # 媒体信息探测
│   └── stream/                        # 摄像头推流脚本
│       ├── run.bat                    # 推流启动脚本
│       └── stream.ps1                 # PowerShell H.264 RTP 推流
│
├── ai_pet/                            # 主程序源码（C++17）
│   ├── CMakeLists.txt                 # CMake 构建配置
│   ├── main.cpp                       # 入口文件（--ui/--chat/--camera/--full + --persona）
│   │
│   ├── ai/                            # AI 模块
│   │   ├── gemma.h / gemma.cpp        # Ollama API 通信（Gemma4 模型调用）
│   │   ├── prompt.h / prompt.cpp      # Prompt 构造（Persona模式/回退模式双通道）
│   │   ├── personality.h / personality.cpp  # 人格进化系统（5级好感度→动态人格）
│   │   ├── strategy.h / strategy.cpp  # 情绪回应策略（4情绪×5等级=20种策略）
│   │   └── persona_loader.h / persona_loader.cpp  # 角色文件加载器（读取 persona.md/memories.md）
│   │
│   ├── vision/                        # 视觉模块
│   │   ├── camera.h / camera.cpp      # 摄像头视频流接收（RTP/H.264 over SDP）
│   │   ├── face.h / face.cpp          # 人脸检测（Haar Cascade）
│   │   └── emotion.h / emotion.cpp    # 表情识别（特征分析器）
│   │
│   ├── memory/                        # 记忆与持久化模块
│   │   └── sqlite.h / sqlite.cpp      # SQLite3 数据库（对话历史/好感度/情绪记录）
│   │
│   ├── controller/                    # 控制器模块
│   │   └── brain.h / brain.cpp        # 主控制器（协调 AI/视觉/记忆/UI 各模块）
│   │
│   ├── ui/                            # UI 模块
│   │   └── ui.h / ui.cpp             # LVGL + SDL2 窗口界面（左右分栏布局）
│   │
│   ├── models/                        # 预训练模型文件
│   │   ├── haarcascade_frontalface_default.xml  # OpenCV 人脸检测模型
│   │   └── emotion-ferplus-8.onnx     # 表情识别 ONNX 模型（备用）
│   │
│   ├── personas/                      # 角色文件目录（ex-skill 生成）
│   │   └── default/                   # 默认内置角色
│   │       ├── persona.md             # 5层人格定义
│   │       ├── memories.md            # 共同记忆
│   │       └── meta.json              # 元数据（昵称、标签等）
│   │
│   ├── third_party/                   # 第三方库
│   │   ├── lv_conf.h                  # LVGL 配置文件（定制化）
│   │   └── lvgl/                      # LVGL 9.2 源码（Git submodule）
│   │
│   └── build/                         # 构建输出目录（CMake 生成）
│       ├── ai_pet                     # 可执行文件
│       ├── data/                      # 运行时数据
│       │   └── memory.db             # SQLite 数据库文件
│       └── models -> ../models/       # 模型目录符号链接
│
└── .gitignore
```

## 模块依赖关系

```
main.cpp
  └── controller/brain
        ├── ai/gemma          ← Ollama HTTP API
        ├── ai/prompt         ← 调用 personality + strategy 或 persona_loader
        │     ├── ai/personality  ← 人格进化（好感度驱动，回退模式）
        │     ├── ai/strategy     ← 情绪回应策略（回退模式）
        │     └── ai/persona_loader ← 5层角色文件加载（Persona模式）
        ├── vision/camera     ← RTP 视频流接收
        ├── vision/face       ← 人脸检测
        ├── vision/emotion    ← 表情识别
        ├── memory/sqlite     ← 对话/好感度/情绪持久化
        └── ui/ui             ← LVGL + SDL2 窗口
```

## 技术栈

| 模块 | 技术 | 版本 |
|------|------|------|
| 构建 | CMake | ≥ 3.16 |
| 语言 | C++17 | GCC 11+ |
| UI | LVGL + SDL2 | 9.2 / 2.0.20 |
| 视觉 | OpenCV | 4.5.4 |
| AI | Ollama (Gemma4) | 本地推理 |
| 存储 | SQLite3 | 3.37+ |
| 字体 | TINY_TTF + Noto Sans CJK | 内置 |
| 推流 | FFmpeg (H.264 RTP) | Windows→WSL |
